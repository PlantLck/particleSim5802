# VS Code Quick Start Guide

**TLDR**: Build and run the particle simulation using Visual Studio Code on Windows in ~60 minutes.

## Prerequisites (What You Need)

1. ‚úÖ Windows 10/11
2. ‚úÖ NVIDIA GPU (RTX 3050 or similar)
3. ‚úÖ ~5GB free space
4. ‚úÖ ~60 minutes time

## 5-Step Installation

### Step 1: Install VS Code (5 min)
1. Download from: https://code.visualstudio.com/
2. Install with default options
3. Open VS Code
4. Install extensions:
   - **C/C++** (Microsoft)
   - **CMake Tools** (Microsoft)

### Step 2: Install Build Tools (15 min)
1. Download: https://visualstudio.microsoft.com/downloads/
2. Get **Build Tools for Visual Studio 2022**
3. Select **Desktop development with C++**
4. Install (~3GB)
5. Restart computer

### Step 3: Install Dependencies (25 min)

**CMake**:
- Download: https://cmake.org/download/ (Windows x64 Installer)
- Install, check "Add to PATH"

**CUDA Toolkit**:
- Download: https://developer.nvidia.com/cuda-downloads
- Select: Windows, x86_64, 10, exe (local)
- Install with defaults (~3GB)

**SDL2 Libraries**:
- Download SDL2: https://www.libsdl.org/download-2.0.php
  - Get SDL2-devel-2.x.x-VC.zip
  - Extract to `C:\SDL2`
- Download SDL2_ttf: https://www.libsdl.org/projects/SDL_ttf/
  - Get SDL2_ttf-devel-2.x.x-VC.zip
  - Extract to `C:\SDL2_ttf`

**Verify** in Command Prompt:
```cmd
cmake --version
nvcc --version
nvidia-smi
```

### Step 4: Setup Project (5 min)
1. Extract project files to folder (e.g., `C:\Projects\ParticleSimulation`)
2. Open VS Code
3. File ‚Üí Open Folder ‚Üí Select project folder
4. Press **Ctrl+Shift+P** ‚Üí Type "CMake: Configure" ‚Üí Enter
5. Select kit: **Visual Studio Build Tools 2022 - amd64**

### Step 5: Build and Run (5 min)
1. Press **Ctrl+Shift+B** (Build)
2. Select **CMake: Build Release**
3. Wait ~2 minutes for compilation
4. Copy DLLs to build folder:
```cmd
copy C:\SDL2\lib\x64\SDL2.dll build\Release\
copy C:\SDL2_ttf\lib\x64\SDL2_ttf.dll build\Release\
```
5. Press **F5** to run!

## Troubleshooting Quick Fixes

**Problem**: CMake can't find SDL2
- **Fix**: Verify `C:\SDL2` and `C:\SDL2_ttf` exist with correct structure

**Problem**: "SDL2.dll not found" when running
- **Fix**: Copy DLLs to `build\Release\` folder (see Step 5)

**Problem**: GPU mode doesn't work
- **Fix**: Verify CUDA installed: `nvcc --version`
- **Fix**: Check GPU drivers updated: `nvidia-smi`

**Problem**: Low performance in GPU mode
- **Fix**: Build in Release (not Debug): Ctrl+Shift+P ‚Üí "CMake: Set Build Type" ‚Üí Release
- **Fix**: Close Chrome/Discord (GPU usage)
- **Fix**: Increase particles with + key (GPU shines with 2000+)

**Problem**: Build fails
- **Fix**: Clean and rebuild:
  - Delete `build` folder
  - Ctrl+Shift+P ‚Üí "CMake: Configure"
  - Ctrl+Shift+B

## Common VS Code Questions

**Q**: Where's the executable?
**A**: `build\Release\particle_sim.exe`

**Q**: How do I debug?
**A**: Press F5, select "Debug Simulation (Windows)"

**Q**: How do I rebuild from scratch?
**A**: Delete `build` folder, then Ctrl+Shift+B

**Q**: Can I build without CUDA?
**A**: Edit `tasks.json`, change `-DUSE_CUDA=ON` to `-DUSE_CUDA=OFF`

**Q**: How do I change CUDA architecture?
**A**: Edit `CMakeLists.txt`, line: `set(CMAKE_CUDA_ARCHITECTURES 86)`
